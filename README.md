# Retinopathy of Prematurity Detection using Deep Learning

## Abstract
This project proposes a deep learning based system for detecting Retinopathy of Prematurity (ROP) using retinal fundus images. The EfficientNet-B3 model is used to classify images into ROP and Non-ROP categories.

## Problem Statement
Manual screening of retinal images requires expert ophthalmologists and is time consuming. This project aims to develop an automated system to assist early detection.
